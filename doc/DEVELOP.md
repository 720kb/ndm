
## Develop ndm

To run the app locally in development mode:

_Setup_

`$ git clone <repo> && cd ndm`

`$ npm install`

_Run app_

#### Run on Linux
`$ npm run linux`

#### Run on Mac
`$ npm run mac`

#### Run on Windows
`$ npm run win`


if you then want to test the executables follow [How to build](https://github.com/720kb/ndm/blob/master/doc/BUILD.md)
